## Gameplay overview

A vertically scrolling shooter for one player. Control an aircraft and dodge or destroy enemy aircraft.

### Player 1 Controls

| Action       | Key          |
|--------------|--------------|
| Move Forward | Up arrow     |
| Move Left    | Left arrow   |
| Move Right   | Right arrow  |
| Move Down    | Down arrow   |
